Regression of daily maximum temperature as collected by NOAA GHCN-Daily.
More stations here. FYI with the stations USW00023293 (San Jose, CA), the sampling takes about 1 minute on my machine. If you don't want to wait, use the fake data provided and just run the sampling.
Raw data
TMAX data
TMAX ~ Normal(alpha + beta * DATE, sigma)
The model is a simple bayesian regression with a linear model and a normal prior (mu=0, sigma=10) on the slope (beta) and the intercept (alpha) and a flat prior on the noise (sigma).
